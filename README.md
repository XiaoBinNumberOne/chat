### 系统使用的技术
1. 后端：Spring Cloud Alibaba、Mybatis-Plus、Springboot、Netty
2. 权限认证：Spring Security、Spring Security Oauth2
3. 网关：Spring Cloud Gateway
4. 数据库：Mysql、Mongodb
5. 中间件：Redis、RocketMQ
6. 存储云：七牛
7. 前端：uniapp、Materialize
8. 同时提供IOS和Android版本
